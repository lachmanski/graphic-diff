Graphic Diff (Coming Soon!)

This tool highlights differences between similar graphics by compositing them with difference blending mode. It enables you to select two local folders on your computer and compare documents between them. Graphic Diff is intended to compare different versions of the same document in order to verify changes made from one version to the next. It supports the formats BMP, GIF, JPG, PDF, PNG, SVG, TIFF, and WebP.
